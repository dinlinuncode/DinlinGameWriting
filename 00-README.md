# 02 课程论文：《两宋都城酒肆与城市社会结构》
宋代都城酒肆的繁荣与城市社会结构息息相关。在空间结构上，酒肆宏观的地理分布受到商业经济，政治权力和城乡关系的影响；微观的建筑结构主要为两层或多层建筑。在人群结构上，酒客分层明显，上层的代表为宗室、富家和士人，推断下层代表为小工商业者；酒人的来源主要有“人力”“街坊”和“闲人”；酒肆主人主要包括大小商人，封建国家和部分官僚。
